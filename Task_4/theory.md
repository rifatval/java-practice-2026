# Работа с базой данных в Java

* Скачать драйвер можно тут - https://jdbc.postgresql.org/download/
* Для того чтобы взаимодействовать с базой данных в Java используется протокол JDBC
* Java Database Connectivity
* Он предоставляет необходимые классы и интерфейсы для работы с БД 
* Все необходимые компоненты находятся в пакете `java.sql`

## Компоненты JDBC

* `DriverManager` - класс, необходимый для создания подключения к базе через JDBC Driver, который он ищет в `classpath`
* `Connection` - интерфейс, представляющий собой `сеанс подключения к базе`
* `Statement` - интерфейс, позволяющий отправлять запросы в базу данных
* `ResultSet` - интерфейс, который содержит итератор по результату запроса

## Как работает JDBC

* Интерфейсы `java.sql` предоставляют описание методов для работы с БД, а конкретные их реализации находятся в конкретном драйвере
* Драйвер менеджер находит нужный драйвер и оттуда предоставляет необходимые классы
* С компонентами JDBC работаем через `try-with-resources`, чтобы каждый компонент автоматически закрывался после использования
